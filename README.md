ΕΡΓΑΣΙΑ: SuperMarketGUI

#1. ΠΕΡΙΓΡΑΦΗ ΕΦΑΡΜΟΓΗΣ
----------------------
Η εφαρμογή SuperMarketGUI είναι ένα γραφικό σύστημα διαχείρισης προϊόντων supermarket
υλοποιημένο σε C# με Windows Forms και βάση δεδομένων MySQL.

Βασικές λειτουργίες:
- Εισαγωγή νέων προϊόντων με όνομα και τιμή.
- Αποθήκευση προϊόντων στη βάση δεδομένων MySQL.
- Προβολή όλων των προϊόντων που υπάρχουν στη βάση.
- Προσθήκη προϊόντων στο «καλάθι» αγορών.
- Υπολογισμός συνολικού κόστους αγορών.
- Διαγραφή προϊόντων από το καλάθι.
- Διαγραφή προϊόντων από τη βάση δεδομένων.
- Ολοκλήρωση αγοράς (Checkout) και εκκαθάριση καλαθιού.
- Χρήση context menus (δεξί κλικ) για διαγραφή στοιχείων.

#2. ΑΝΑΛΥΤΙΚΗ ΠΑΡΟΥΣΙΑΣΗ ΚΛΑΣΕΩΝ
------------------------------

2.1 Κλάση Product
-----------------
Σκοπός:
Αναπαριστά ένα προϊόν του supermarket.

Πεδία / Ιδιότητες:
- Id : μοναδικός κωδικός προϊόντος.
- Name : όνομα προϊόντος.
- Price : τιμή προϊόντος.

Μέθοδοι:
- Product(int id, string name, double price)
  Constructor που αρχικοποιεί τα δεδομένα του προϊόντος.

- ToString()
  Επιστρέφει μορφοποιημένη συμβολοσειρά με όνομα και τιμή του προϊόντος.

2.2 Κλάση Form1
---------------
Σκοπός:
Αποτελεί το κύριο παράθυρο της εφαρμογής και διαχειρίζεται
όλη τη λειτουργικότητα του συστήματος.

Βασικά πεδία:
- products : λίστα προϊόντων καλαθιού.
- prices : λίστα τιμών καλαθιού.
- dbProducts : λίστα προϊόντων που φορτώνονται από τη βάση.
- total : συνολικό ποσό αγοράς.
- cmsList : context menu για το καλάθι.
- cmsDb : context menu για τα προϊόντα της βάσης.
- connStr : στοιχεία σύνδεσης με MySQL.

Κύριες μέθοδοι:

- Form1()
  Αρχικοποιεί το περιβάλλον χρήστη, τα context menus,
  φορτώνει τα προϊόντα από τη βάση και ενημερώνει το σύνολο.

- LoadFromDatabase()
  Φορτώνει όλα τα προϊόντα από τον πίνακα products
  και ενημερώνει τη λίστα της βάσης.

- btnAdd_Click()
  Εισάγει νέο προϊόν.
  Πραγματοποιεί ελέγχους εγκυρότητας και αποθηκεύει το προϊόν στη βάση.

- btnDelete_Click()
  Διαγράφει προϊόν από το καλάθι.
  Προαιρετικά διαγράφει και το αντίστοιχο προϊόν από τη βάση.

- btnDeleteDB_Click()
  Διαγράφει επιλεγμένο προϊόν απευθείας από τη βάση δεδομένων.

- btnCheckout_Click()
  Εμφανίζει το συνολικό ποσό αγοράς και αδειάζει το καλάθι.

- btnLoad_Click()
  Επαναφορτώνει τα δεδομένα από τη βάση.

- listBoxDB_DoubleClick()
  Προσθέτει προϊόν από τη βάση στο καλάθι με διπλό κλικ.

- ListBox_MouseUp()
  Διαχειρίζεται την εμφάνιση των context menus.

- DeleteCart_Click()
  Διαγραφή προϊόντος από το καλάθι μέσω context menu.

- DeleteDb_Click()
  Διαγραφή προϊόντος από τη βάση μέσω context menu.

- button1_Click()
  Ανανέωση της λίστας προϊόντων της βάσης.

2.3 Κλάση Program
-----------------
Σκοπός:
Αποτελεί το σημείο εκκίνησης της εφαρμογής.

Μέθοδοι:
- Main()
  Εκκινεί την εφαρμογή Windows Forms και εμφανίζει τη Form1.

#3. ΟΔΗΓΙΕΣ ΕΓΚΑΤΑΣΤΑΣΗΣ ΚΑΙ ΕΚΤΕΛΕΣΗΣ
-------------------------------------

Απαιτήσεις:
- Windows 10 ή νεότερα.
- .NET 6 SDK.
- MySQL Server.
- Visual Studio 2022.

Βήματα:

1. Δημιουργήστε βάση δεδομένων MySQL.
2. Εκτελέστε το αρχείο sql.sql.

Περιεχόμενο SQL:
CREATE DATABASE supermarket;
USE supermarket;

CREATE TABLE products (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    price DOUBLE
);

3. Ελέγξτε ότι τα στοιχεία σύνδεσης στη μεταβλητή connStr
   είναι σωστά για τον MySQL Server σας.

4. Ανοίξτε το αρχείο SuperMarketGUI.sln στο Visual Studio.

5. Επιλέξτε Build -> Build Solution.

6. Εκτελέστε την εφαρμογή με F5.

#4. ΟΔΗΓΙΕΣ ΕΠΙΔΕΙΞΗΣ ΕΦΑΡΜΟΓΗΣ
------------------------------

Σενάριο επίδειξης:

1. Εκκίνηση εφαρμογής.
2. Εισαγωγή ονόματος προϊόντος και τιμής.
3. Επιλογή Add για αποθήκευση.
4. Εμφάνιση του προϊόντος στη λίστα της βάσης.
5. Διπλό κλικ σε προϊόν της βάσης για προσθήκη στο καλάθι.
6. Παρατήρηση της ενημέρωσης του συνολικού ποσού.
7. Διαγραφή προϊόντος από το καλάθι.
8. Διαγραφή προϊόντος από τη βάση με το κουμπί Delete DB ή δεξί κλικ.
9. Επιλογή Checkout και εμφάνιση συνολικού ποσού αγοράς.
10. Επιβεβαίωση ότι το καλάθι αδειάζει μετά την ολοκλήρωση.
